GitHub Gist Sync Setup

Flashy uses GitHub Gist as a free cloud sync backend. Your data is stored in a private Gist only you can access.

Generate at github.com/settings/tokens → Fine-grained tokens → Set Gists: Read and write
⚠ Images embedded in cards are stored as base64 and can make the Gist large. If sync fails, export a JSON backup first.
